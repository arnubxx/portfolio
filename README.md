# Arnub Datta - Portfolio

A modern portfolio website built with Next.js, React, and Tailwind CSS.

## Features

- Responsive design with Tailwind CSS
- Fast performance with Next.js
- Dark theme with modern aesthetics
- Project showcase
- Skills section
- Research experience
- Contact information

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

Build for production:

```bash
npm run build
```

### Production

Start the production server:

```bash
npm start
```

## Deployment on Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com?utm_medium=default-template&utm_source=create-next-app&utm_campaign=create-next-app-readme).

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Sign up or log in with GitHub
4. Click "Add New..." → "Project"
5. Import your GitHub repository
6. Click "Deploy"

Your portfolio will be live on a Vercel URL!
